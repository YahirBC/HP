II. INFORMACIÓN MÁS RECIENTE

PROBLEMAS CONOCIDOS Y SOLUCIONES

General

 

· Algunas tarjetas USB 2.0 de alta velocidad se envían con controladores de host sin firmar. Si ha adquirido una tarjeta complementaria USB 2.0 y dispone de un controlador sin firmar, es posible que tenga problemas con la instalación del controlador o con la comunicación entre el dispositivo y el software de HP. Se recomienda utilizar siempre un controlador de host con la certificación Microsoft WHQL con el hardware USB.

·         Los paquetes de controladores de impresión/escaneado Plug and Play están disponibles en la Web en

las siguientes direcciones URL:

www.hp.com/support/ljm1210series

www.hp.com/support/ljm1130series

·         Se limitará la funcionalidad de los sistemas que sólo cumplan los requisitos mínimos. El uso de muchos vídeos o imágenes requiere un sistema que cumpla los requisitos recomendados del sistema para ofrecer una fiabilidad y un rendimiento aceptables.

·         Resolución de problemas en la configuración y la instalación

Si no se reconoce el dispositivo:

  Compruebe que el cable es compatible con USB 2.0 de alta velocidad.

  Asegúrese de que el cable está conectado correctamente. Extraiga el cable USB, compruebe    el extremo del cable y, a continuación, hágalo coincidir con el patrón del puerto USB. Asegúrese de que no está al revés. Introduzca completamente uno de los extremos del cable USB

  en el puerto USB del dispositivo HP, y el otro extremo en la parte posterior del ordenador.

  Compruebe que el periférico está encendido. Los periféricos no se reconocen si están apagados.

  Asegúrese de que el cable USB no mide más de 5 metros (15 pies) de longitud.

  Compruebe que el puerto USB está habilitado. Algunos equipos se suministran con los puertos USB deshabilitados.

  Si utiliza una estación de conexión o un replicador de puertos con un portátil  , intente conectar el cable USB a un puerto en vez de al portátil.

  Algunos PC/portátiles no tienen suficiente energía para los puertos USB.  Intente utilizar un hub que reciba energía del USB si todavía no reconoce el dispositivo y si los pasos anteriores no resuelven el problema.

· En un sistema Thai, algunos componentes pueden mostrar texto en inglés.

Escaneo

·         El escaneado con resoluciones más altas puede llevar mucho tiempo y el proceso creará archivos grandes. Para la mayoría de objetivos, la resolución predeterminada proporciona el mejor equilibrio entre calidad y tiempo o tamaño.  También el escaneo en red puede llevar mucho tiempo.

·         Cuando se escanea en un archivo, el tipo de archivo predeterminado para el software de escaneado HP LaserJet Professional Scan es JPEG y cuando se escanea en un correo electrónico, el tipo de archivo predeterminado es PDF. Los archivos JPEG son compatibles con un tipo de salida de 256 tonos de gris (8 bits). Si selecciona un tipo de salida en blanco y negro (1 bit), el archivo producido conserva el formato

JPEG. Para obtener un tipo de salida en blanco y negro, cambie la configuración del tipo de salida predeterminado a TIF o BMP en el cuadro de diálogo Guardar configuración del menú Configuración de imagen de escaneado, al que ha accedido mediante el menú Configuración del software de HP LaserJet Professional Scan para el dispositivo.

·         Asegúrese de que la tapa está correctamente centrada en el escáner para evitar que entre luz en el escaneado y obtener así los mejores resultados de recorte automático. Es posible que algunas imágenes no se recorten automáticamente de forma correcta incluso cuando la tapa está correctamente colocada. En estos casos, tendrá que ajustar de forma manual el área de recorte.

·         Recomendaciones de OCR para los archivos de entrada:

  300 ppp, escala de grises de 8 bits o datos de imagen de 24 bits.

  400 ppp, escala de grises de 8 bits o datos de imagen de 24 bits para idiomas asiáticos (chino tradicional y simplificado, japonés y coreano).

  No utilice archivos de 1 bit para OCR.

·         Si se colocan algunas hojas de papel encima de los originales escaneados de superficie plana, esto puede evitar que se corra el color en la imagen de la segunda cara o que se forme espuma y contribuir a alisar el original.

·         Los originales doblados, arrugados o marcados pueden hacer que aparezcan sombras en la imagen.

Fax

·     Dependiendo de la calidad de la línea de telecomunicación, es posible que los faxes transmitidos o recibidos con la velocidad máxima del fax (V.34) fallen.  Si se producen errores de transmisión/recepción, cambie la velocidad del fax a Media (V.17) o Lenta (V.29) en el menú Panel de Control del dispositivo y vuelva a intentarlo.  Este producto sólo admite

líneas de fax analógicas.  Si se utiliza una línea de fax digital VOIP (voz sobre IP), PBX o DSL, póngase en contacto con el proveedor de la línea digital para comprobar la compatibilidad si se producen problemas de envío/recepción de fax.

Una vez finalizada la actualización del sistema operativo, vuelva a instalar el dispositivo HP con el software más reciente desde la siguiente

III. INSTRUCCIONES DE INSTALACIÓN

Cuando se utilice en un entorno de Windows, el software de instalación o configuración del CD-ROM detectará la versión y el idioma del sistema operativo que esté ejecutando.

INSTALACIÓN DE SOFTWARE EN SISTEMAS WINDOWS XP, Windows Vista de 32 bits y Windows 7

NOTA: Para instalar el sistema de impresión en un sistema operativo Windows XP, Windows Vista o Windows 7, es necesario disponer de privilegios de administrador .

Instalación de un sistema de impresión desde un CD-ROM

1.       No conecte el cable USB al ordenador hasta que se lo indique el instalador del software.

2.       Cierre todos los programas.

3.       Inserte el CD-ROM.

4.       Si el programa instalador del sistema de impresión de HP no ha empezado después de 30 segundos, complete los pasos del 5 al 7.

5.       Seleccione Ejecutar del menú Inicio de la barra de tareas.

6.       Desplácese al directorio raíz de la unidad de CD-ROM.

7.       Seleccione el archivo Setup.EXE, haga clic en Abrir y, a continuación, haga clic en Aceptar.

8.       Siga las indicaciones.

DESINSTALADOR

Cuando haya instalado un sistema de impresión, puede utilizar el icono de desinstalación del grupo de programas HP LaserJet Professional serie M1210 y M1130 MFP para eliminar todos los componentes del sistema de impresión HP.

Inicio del desinstalador

1.       Desde el menú Inicio, en el grupo de programas HP LaserJet Professional serie M1210 o M1130 MFP, haga clic en el icono de desinstalación de HP LaserJet Professional serie M1210 y M1130 MFP.

2.       El sistema de impresión le guía durante la eliminación de todos sus componentes.

 

NOTA: 

Si ha llevado a cabo una instalación típica y no tiene ningún otro dispositivo HP instalado, también tendrá que seguir los pasos 1 y 2. No siga estos pasos si tiene otros dispositivos HP instalados: podrían afectar en parte a la funcionalidad del resto de dispositivos HP, ya que comparten el software.

 

1.       Abra el Panel de control de Microsoft Windows. Haga doble clic en Agregar o quitar programas.

2.       Verá entradas para Funciones ampliadas de HP y para Actualización de software de HP.

Para cada elemento, haga clic en Cambiar o quitar y siga las instrucciones.
